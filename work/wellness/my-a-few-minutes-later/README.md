# my-a-few-minutes-later

안드로이드 (특히 갤럭시) 유저가 사용하는 기본 시계 애플리케이션은
절대 시간 기반의 알람 (실제 구체적인 시각을 입력하여 알람을 생성하는 방식) 과 
상대 시간 기반의 타이머 (원하는 시간을 입력하여 타이머로 남은 시간을 카운트 다운 하는 방식) 만 제공하는데 

이동시간에 밀린 잠을 보충하는 나는
지금으로부터 n분 후 울릴 알람을 생성해야하는 경우가 많음..

n분 후처럼 상대 시간 기반으로 알람을 생성하려면
갤럭시에서는 gemini나 bixby에게 부탁하는 방법 뿐인데 

Android에서는 AlarmClock.ACTION_SET_ALARM intent를 직접 접근해서 
내가 원하는 시간 후에 울릴 알람이 생성되도록 하는
a few minutes later 앱 apk를 만들어봄!!
