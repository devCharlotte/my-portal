# Summer Routine

Coursework와 TA 업무 종강 이후의 여름 루틴 페이지입니다.

## Routine Alarm

- Weekly timetable
- Desktop Notification
- Android에서는 오늘 요일에 해당하면서 현재 시각 이후인 루틴만 표시
- 각 루틴을 Galaxy 시계에 반복 요일 없는 일회성 알람으로 개별 추가
- 페이지 입력값을 GitHub 새 Issue 화면으로 자동 전달
- 저장소 소유자가 `Submit new issue`를 누르면 GitHub Actions가 `routines.json`을 자동 수정·커밋하고 Pages를 재배포
- 루틴 이름·시간·요일은 필수, 알림 내용은 선택

## Pilates & Yoga Sequence Studio

- Yoga, Pilates, Recovery Ring, Magic Circle 동작 205개
- 자동 시퀀스 생성과 quick preset
- 검색, 신체 부위, Yoga style, 카테고리, 즐겨찾기
- 상세 가이드와 Teaching Tip
- 순서·수행시간 편집과 작업 중 자동 복원
- 저장·불러오기, Timer, Voice countdown, Breath guide
- Workout calendar와 streak
- 1080×1920 Story image 생성·다운로드·공유
