# Summer Routine & Yoga Sequence

Wellness의 Routine Alarm과 Pilates & Yoga Sequence Studio를 통합한 페이지입니다.

## 유지된 Routine 기능

- Weekly timetable
- Desktop Notification
- 오늘 요일·현재 시각 이후 항목만 Galaxy Clock 일회성 알람으로 추가
- Android 반복 요일 미전달
- 관리자 비밀번호 검증 후 routines.json GitHub commit

## Sequence Studio 기능

- 205개 Yoga, Pilates, Recovery Ring, Magic Circle 동작
- 자동 시퀀스 생성과 5개 quick preset
- 검색, 신체 부위, Yoga style, 카테고리, 즐겨찾기
- 상세 가이드와 Teaching Tip
- 순서 변경, 동작 시간 변경, 현재 작업 자동 복원
- 저장·불러오기, Timer, Voice countdown, Breath guide
- Workout calendar와 streak
- 1080×1920 Story image 생성·다운로드·공유
