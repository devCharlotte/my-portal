# my-tab-customizer

한 번에 너무 많은 창을 분할해서 띄우고 </br>
분할한 창에도 여러 탭이 있어서 너무 불편했음... </br>

현재 Chrome 탭의 이름이랑 아이콘 바꾸기! </br>
카테고리별로 탭 그룹화하기!</br>

## 기능

- 탭 별칭 변경
- 현재 사이트의 기존 아이콘 유지
- JH 아이콘, 이미지 URL 또는 이미지 파일을 탭 아이콘으로 적용
- 자주 사용하는 탭 카테고리 저장 선택 삭제
- 같은 카테고리의 탭을 한 그룹으로 병합
- 확장 프로그램이 변경한 탭 이름 아이콘 그룹 상태 복원

## 설치 또는 업데이트

1. 깃허브에서 레파지토리를 다운 받고 ZIP 압축 풀기
2. Chrome 주소창에 `chrome://extensions` 입력
3. 개발자 모드 켜고
54. **압축해제된 확장 프로그램을 로드합니다**를 눌러서
6. 압축을 푼 폴더 안의 `my-tab-customizer` 폴더 선택하기
